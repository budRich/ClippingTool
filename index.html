ClippingTool, the saga:

My dayjob consist of copy/pasting stuff from and to inputboxes. Ctrl+C, Ctrl+V isn't enough, i needed more clipboards. I have used various clipboard manager for personal use for years. But at work i needed a clipboard manager that was more strict. Also, mine and many other workplaces have strict IT rules. To have a clipboard manager approved i had to be sure it didn't save sensetive data anywhere.

This is ClippingTool

First time you launch clipping tool a windo appears on the left side of the screen. Some inputboxes in the window is numbered 0-9.

Box 1-9 are the clipboards for this case. At the top of the window is the tab bar, each tab is a case. Box 0 is the command box, from here you can search your notes. Below box 0 is the current note, to the right of that box is a filtered list of notes.

When ClippingTool window is active you can use the following hotkeys:

Alt+[0-9]: Focus box[0-1]
Ctrl+S: Save current note.
Ctrl+T: New tab.
Ctrl+W: Close tab.
Ctrl+[1-9]: GoTo tab[1-9].

The hotkeys described below are global and works even if ClippingTool window is hidden.

CapsLock: Show/Activate ClippingTool window, if it is active, activate the previously activated window.
Ctrl+CapsLock: Hide ClippingTool window.
CapsLock+[0-9]: Send content of inputbox[0-9].
Shift+CapsLock+[0-9]: Set content of inputbox[0-9] to selected text.
CapsLock+`: Send note.
Shift+CapsLock+`: Set note to selected text.

Profiles and notes

When you save a note you can choose to make it either global or local.

If it's local it will be saved in ClippingTool\profiles\%profile%\%folder%\%note% . %profile% %folder% and %note% is set by you. Locally saved notes can only be viewed when it's corresponding profile is open.

If it's global it will be saved in ClippingTool\global\%folder%\%note% . %folder% and %note% is set by you. Global notes can be accessed from any profile.

If a note contains a string thats equal to a box, it will be converted to a template when saved. This means when saving a note that includes "LeifLoket", that happens to be whats in box2, in this profile: username. Next time this template is loaded, the value of username will be where the original note said: "LeifLoket".

A profile is basicly a list of what boxes is to be visible in the window.

This is how you create a profile:

  • Create a folder in ClippingTool\profiles\ , the name of the folder is the name of the profile.
  • Create a file named list in this folder.
  • Every line in list, is the name of the corresponding input box.
  • You can set the current profile from the window menu.
  • Hold shift when you create a new tab, to choose profile for the tab.
